# NAC-Niyonne-Anti-Cheat-
NAC is an anti-cheat system based on real-time game data. It analyzes player behavior and prevents bypass of NAC flags and NAC bans (temporary or permanent). Built in GDScript, version 1.1.2. Active on Button Breaker; planned for Angels Hospital. This Anti-cheat is generalized, that is to say that its code is different depending on the game
This anti-cheat is a work in progress. Feedback, code review, and vulnerability reports are welcome.
